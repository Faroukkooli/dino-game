# Dino Game Rip off

This project is written in C and uses [CMake][] as a build system.<br>

It is a simple rip off of the famous dino game on the Chrome browser using the [SDL2][SDL] Library.

## Dependencies

- [Git][]
- C Compiler (gcc, ...)
- [CMake][]
- [SDL2][SDL] library
- [SDL2_image][] library
- [SDL2_gfx][] library
